**Kuray's Overworld Encounters!**

**Please note:** Mod menu needed to be edited a little bit, however, it should work the same. You may need to replace your mod menu with my copy. (sorry @stonewall)

This mod spawns wild Pokemon directly in the overworld that you can see and interact with! All features are toggle-able and customizable through Mod Settings.

**Features List**

**Overworld Pokemon Spawning** - Wild Pokemon appear on the map! Spawn on grass, water (when surfing), and caves based on encounter tables.

**Disable Encounters** - Toggle to disable overworld encounters entirely.

**Shiny Rate** - Adjust shiny rate from 1/1 to 1/65536 (default: 1/8192).

**Spawn Distance** - Control despawn distance (1-20 tiles, default: 12).

**Despawn Far Events** - Auto-despawn Pokemon that move too far away for performance.

**Bright Shinies** - Shinies stay bright during night/dark areas, unaffected by day/night tone.

**Colorful Names** - Pokemon names colored by gender (blue=male, red=female).

**Water Spawns (Surf Only)** - Water Pokemon only spawn when surfing.

**Spawn & Shiny Animations** - Customize animation IDs (default: 2 spawn, 53 shiny).

**Despawn Shinies** - Toggle if shinies can despawn (default: protected).

**Different Encounters** - Use alternative encounter tables for variety.

**Encounter Table ID** - Select encounter table index (0-10).

**Max Encounters** - Max Pokemon per map (1-20, default: 5).

**Fusion Encounters** - Enable wild fusion Pokemon! Appear as dark silhouettes until interacted with.

**Fusion Rate** - Control fusion spawn rate (1/1 to 1/100, default: 1/10).

**Horde Battles (2v1)** - Nearby Pokemon team up for double battles!

**Horde Distance** - Distance threshold for horde battles (1-5 tiles, default: 2).

**Spawn on Map Load** - Spawn multiple Pokemon immediately when entering maps.

**Initial Spawn Count** - Pokemon count on map load (1-15, default: 6).

**Nature-Based Behavior** - Pokemon move based on nature! Timid flee, Brave charge, Curious approach, etc.

**Species-Specific Behavior** - Custom behaviors for certain Pokemon (Wimpod flees, Veluza chases, etc.).

**Shiny Detection** - Shinies play sound effects and sparkles when spawning.

**Log Spawns** - Debug option to log spawns to console.

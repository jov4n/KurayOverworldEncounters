# Kuray's Overworld Encounters

A Pokemon Essentials mod that spawns wild Pokemon directly in the overworld that you can see and interact with!

## Description

This mod spawns wild Pokemon directly in the overworld that you can see and interact with! All features are toggle-able and customizable through Mod Settings.

**Requires:** Mod Settings

**Credits:** Based on Voltseon's Overworld Encounters, adapted by Fill to include fusion encounters, horde battles, and compatibility with Kuray's Infinite Fusion.

## Installation

1. Download the mod files
2. Place the `Mods` folder contents into your game's `Mods` folder
3. Place the `Graphics` folder contents into your game's `Graphics` folder
4. Replace your Mod Settings file with the included version (see note below)

**Please note:** Mod menu needed to be edited a little bit, however, it should work the same. You may need to replace your mod menu with my copy. (sorry @stonewall)

## Features

- **Overworld Pokemon Spawning** - Wild Pokemon appear on the map! Spawn on grass, water (when surfing), and caves based on encounter tables.
- **Shiny Rate Customization** - Adjust shiny rate from 1/1 to 1/65536 (default: 1/8192).
- **Fusion Encounters** - Enable wild fusion Pokemon! Appear as dark silhouettes until interacted with.
- **Horde Battles (2v1)** - Nearby Pokemon team up for double battles!
- **Nature-Based Behavior** - Pokemon move based on nature! Timid flee, Brave charge, Curious approach, etc.
- **Species-Specific Behavior** - Custom behaviors for certain Pokemon (Wimpod flees, Veluza chases, etc.).
- And many more customizable features!

See `DISCORD_RELEASE.md` for the full feature list.

## Configuration

All features can be customized through the Mod Settings menu under the "Encounters" category. Most features are enabled by default for the best experience.

## Requirements

- Mod Settings
- Kuray's Infinite Fusion (for fusion encounters)
